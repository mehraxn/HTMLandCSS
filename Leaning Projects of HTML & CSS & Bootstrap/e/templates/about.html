Concept of Route 2

This page exists to explain the route /about.

What happens when you open /about ?

  1. The browser requests the URL /about.
  2. Flask looks for a matching route.
  3. Flask finds @app.route("/about").
  4. Flask runs the function about().
  5. The function returns about.html.
  6. The browser displays this page.

Meaning of this route

/about is usually used for information about the site, app, or company.

Mapping:
/about → about() → about.html

How is it different from the first route?

This route is not the homepage. Because the path is different, Flask does not run home(). It runs about() instead.
